Beleriand

Your task is to implement Beleriand RPG game. In the game, Legolas is in Beleriand full of orks. Legolas has to destroy the orks before his elfbread runs out and the orks can find and kill him. Legolas can't see the orks while he is running, so he is blind until he stops. With one move, Legolas can run as many steps as he wants, but he has to eat one elfbread after each run (not step! A run consists multiple steps).

The game situation, i.e. Beleriand, Legolas and the orks are shown in text form. The first line in the print output tells how many runs Legolas has left (that is to say, how much elfbread has left). After that, the print output shows Legolas and ork positions, which in turn are followed by the game map. In the example below, you see Legolas (@) and three orks (o); in this case, Legolas has enough elfbread for fourteen moves.

The user (Legolas) can move by giving a sequence of commands and pressing Enter. The commands are:

  • w - go up
  • s - go down
  • a - go left
  • d - go right

When the user commands are executed (the user can give many commands at once), a new game situation is drawn. If the elfbread reaches 0, the game ends and the text YOU LOSE is printed on the board.

The orks move randomly in the game, and they take one step for each step Legolas takes. If Legolas and a ork run into each other (even momentarily) the ork is destroyed. If an ork tries to step outside the board, or into a place already occupied by another ork, the move is not executed. When all the orks are destroyed, the game ends and it prints YOU WIN.

In order to help the tests, create the class Beleriand in your game, with:

  • the constructor public Beleriand(int length, int height, int orks, int moves, boolean orksMove)

    the values length and height represent the dimension of Beleriand (always a rectangle); orks stands for the initial number of orks (the positions of the orks can be decided randomly); moves determines the initial number of moves; and if orksMove is false, the orks do not move.

  • the method public void run(), which starts the game

Attention! Legolas starts the game in the position 0,0!

Attention! Legolas and the orks can not move out of Beleriand and two orks cannot step into the same place!
